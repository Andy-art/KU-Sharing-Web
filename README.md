# KU Sharing(Web)
- 주제 : 건국대학교 글로컬캠퍼스 학생들이 서로 물건을 자유롭게 공유, 기부할 수 있는 서비스
- 프로젝트 기간 : 2019.05~2019.06
- 팀원 : 석정환, 장용운

# 기술 스택
- HTML, CSS, PHP, Apache, MySQL

# 프로젝트 내용
- HTML, CSS로 기초적인 회원가입, 공유, 기부, 검색 기능 구성
- XAMPP(Apache, MySQL, PHP)를 사용하여 웹서버와 연동
- 인터페이스 기본설계 원칙 CRUD 설계 구현
